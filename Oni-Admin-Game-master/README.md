# Oni-Admin-Game
An admin game where you control a Bus Company. This game is being made for "Workshop" , a school subject where we develop a game with all 3 specialization course of the school in a group.

# Engine 
Unity 3D
# Genre 
Bussines Game
# Description
A bussines game where the player manages a bus company of a typical brazilian city from the beginning with few bus until manage the most powerful consortium of the city.
